# Envy E-commerce

Envy E-commerce is a modern and responsive e-commerce web application built with **Vite** and **ReactJS**. It features a variety of pages including home, product collection, product details, cart, and more. The styling is powered by **TailwindCSS** for a sleek, utility-first design.

---

## ✨ Features

*   **User Authentication:** Secure login functionality for users.
*   **Product Discovery:** Comprehensive product listing and detailed view pages.
*   **Shopping Cart:** Fully functional cart for managing selected items.
*   **Checkout System:** Streamlined order placement process.
*   **Responsive UI:** Mobile-first design ensures a great experience on all devices.
*   **Toast Notifications:** Real-time feedback using React Toastify.

## 🚀 Installation & Setup

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/rudra-xi/envy-ecommerce.git](https://github.com/rudra-xi/envy-ecommerce.git)
    ```

2.  **Navigate to the project directory:**
    ```bash
    cd envy-ecommerce
    ```

3.  **Install the dependencies:**
    ```bash
    npm install
4. **Start the development server:**
     ```bash
     npm run dev
> [!TIP]
> Once the server is running, open your browser and navigate to **http://localhost:8000** to view the app.
